# 📊 Análisis de Ventas

Este proyecto realiza un **análisis exploratorio de datos de ventas** utilizando Python y librerías de análisis de datos.  
El objetivo es responder algunas preguntas clave sobre el comportamiento de las ventas a partir de gráficos y estadísticas.

El dataset utilizado fue obtenido desde **Kaggle** y se llama **SuperStoreOrders**, el cual contiene información sobre productos, ciudades, cantidades y ventas realizadas.

---

# 📌 Preguntas y Respuestas

## 1️⃣ ¿Qué producto genera mayores ventas?

El producto que genera mayores ventas es **"Eldon File Cart, Single Width"**, según el análisis mostrado en el **gráfico de barras**, donde se observa que este producto tiene el mayor volumen de ventas en comparación con los demás.

---

## 2️⃣ ¿Qué ciudad tiene mayor volumen de ventas?

La ciudad con **mayor volumen de ventas dentro del Top 10** es **California**, de acuerdo con el análisis realizado sobre las ventas por ubicación.

---

## 3️⃣ ¿Cuál es el promedio de ventas?

El **promedio de ventas** calculado a partir de los datos es:

```
127.29505868590368
```

Este valor representa el **promedio de ventas por transacción** dentro del conjunto de datos analizado.

---

## 4️⃣ ¿Existe mucha variación en las ventas?

Sí, existe **una alta variación en las ventas**.

Según el **histograma de ventas**, se observa que:

- La mayoría de las transacciones se concentran en **valores bajos de ventas**.
- Existen **algunas transacciones con valores muy altos**.
- Esto genera una **distribución asimétrica hacia la derecha**.

---

## 5️⃣ ¿Qué producto debería promocionarse más según los datos?

Según el análisis de los gráficos, especialmente el **gráfico de barras**, los productos relacionados con **carpetas o "Binders"** presentan un alto volumen de ventas.

En particular, el producto:

**"Eldon File Cart, Single Width"**

es uno de los **más vendidos**, por lo que **debería promocionarse más** para potenciar aún más las ventas.
